

# STUDENT ROSTER MANAGEMENT SYSTEM

## Overview 
This project is a C++ console-based application. It manages a class roster of students by storing, displaying, and manipulating student records using object-oriented programming principles.

The system reads a predefined set of student data, parses each record, and stores it in a roster. It then provides functionality to display all students, validate email addresses, calculate average days spent in courses, and filter students by degree program.

## System Functionality
The application performs the following core operations:

Parses student data strings and adds them to a roster
Stores student objects in a dynamically managed data structure
Displays all student records in a formatted table
Validates and identifies invalid email addresses based on formatting rules
Calculates and displays the average number of days a student spends in three courses
Filters and displays students based on degree program (SECURITY, NETWORK, SOFTWARE)
Removes a student from the roster by student ID

## Key Concepts Demonstrated 
Object-Oriented Programming (OOP)
Class design and encapsulation
Inheritance and polymorphism (student types by degree program)
Pointer-based memory management
Dynamic array management
String parsing and validation logic
Modular program architecture (separate header/source files)
